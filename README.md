# Data Quality Assessment System for Philippine Education Open Datasets

## Problem Statement

I want to answer: **"How can a data quality assessment system identify and measure data quality issues in Philippine education open datasets to improve their reliability and usability for data analysis and decision-making?"**

## Audience

This project is for **education researchers, school administrators, government agencies (DepEd, CHED, and PSA), policymakers, and data analysts** who use Philippine education datasets for research, planning, and decision-making.

## KPI or Key Metric

The main metrics I want to track are:

* **Data Quality Score**
* **Data Completeness Rate**
* **Data Consistency Rate**
* **Duplicate Record Count**
* **Invalid Record Count**
* **Data Accuracy Rate** (based on validation rules)

## Likely Data Source

I will explore publicly available education datasets from:

* Department of Education (DepEd) Open Data
* Commission on Higher Education (CHED) Open Data
* Philippine Statistics Authority (PSA) Education Statistics
* Philippine Open Data Portal (https://data.gov.ph/)

## Possible Final Dashboard

The dashboard should help the audience quickly see:

* Overall data quality score of each education dataset
* Missing, duplicate, and invalid records
* Data quality comparison across different datasets
* Summary of validation results
* Which datasets are ready for analysis and which require further cleaning
